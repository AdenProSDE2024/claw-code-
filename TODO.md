## 格式规范

每个任务一行,使用 Markdown 复选框:

```markdown
- [ ] 任务描述 @due(2026-07-15) @prio(P0)
- [x] 已完成的任务 @due(2026-07-10) @prio(P1)
```

| 标记 | 含义 | 说明 |
|------|------|------|
| `- [ ]` | 未完成 | Mentor 会持续监督 |
| `- [x]` | 已完成 | Mentor 会表扬你 |
| `@due(YYYY-MM-DD)` | 截止日期 | 没写的话 Mentor 会催你定一个 |
| `@prio(P0/P1/P2)` | 优先级 | P0 最高;同时进行的 P0 太多会被提醒聚焦 |

分区(`## 进行中` 等)只是给人看的,Mentor 会扫描全文件的所有任务行。
