## OpenClaw Installation

[OpenClaw](https://github.com/openclaw/openclaw) is an open-source personal AI assistant that runs on any OS and platform.

### Prerequisites

- Node.js >= 22

### Install via npm

```bash
npm install -g openclaw@latest
```

### Verify installation

```bash
openclaw --version
```

### Getting started

After installation, run the onboarding command:

```bash
openclaw onboard --install-daemon
```

For more details, see the [official installation docs](https://docs.openclaw.ai/install).
